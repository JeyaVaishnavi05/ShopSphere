# 🛒 ShopSphere – Dynamic E-Commerce Platform

ShopSphere is a modern e-commerce web application designed to simulate a real-world online shopping experience. The platform allows users to browse products, manage their shopping cart, and complete the checkout process, while providing an admin dashboard for efficient product management and store operations.

The project focuses on building a dynamic shopping platform with database integration, CRUD operations, and interactive data visualization to understand the core workflow of modern e-commerce applications.

---

## Features

### Customer Features
- Dynamic product listing with product cards
- Browse and explore available products
- Add products to shopping cart
- Manage cart items and quantities
- Checkout functionality for order placement
- Responsive user interface
- Dark mode support for enhanced user experience

### Admin Dashboard
- Add new products
- Edit existing product details
- Delete products from inventory
- Search and manage products
- Store product information dynamically using MongoDB
- View sales insights through interactive charts
